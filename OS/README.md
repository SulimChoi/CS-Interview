## 1. Introduction   
  - __Operating System__   
    - 컴퓨터의 하드웨어를 관리하는 프로그램   
    - 컴퓨터 사용자와 하드웨어 사이의 중개자 역할을 하며 응용프로그램을 위한 기초를 제공
